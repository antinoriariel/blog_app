## Hola

Soy alguien que escribe sobre tecnología, código y los proyectos que voy armando. Este blog existe para no perder lo que aprendo en el camino.

## De qué trata el blog

Principalmente de desarrollo de software: Python, herramientas, arquitectura de sistemas pequeños y medios, y cualquier cosa que me parezca lo suficientemente interesante como para documentar.

No hay un tema único ni una frecuencia fija. Escribo cuando tengo algo que decir.

## Cómo está hecho

Este sitio es completamente estático. No hay base de datos, servidor de aplicaciones ni JavaScript del lado del servidor. Solo HTML, CSS y un generador Python que convierte archivos Markdown en páginas.

El código está disponible en el repositorio del proyecto si querés ver cómo funciona o usarlo como base para tu propio blog.
